# Food Dataset Management – Kota Malang

This project contains manually collected food and restaurant data from Google Maps in Malang City, Indonesia.

## Project Overview

The dataset was created as a personal data management and data entry project to practice structured data collection, spreadsheet organization, and data validation.

The project contains:
- 372 food menu records
- 89 restaurants and food stalls
- Structured spreadsheet data using Microsoft Excel and Google Sheets

---

## Project Activities

- Manual data collection from Google Maps
- Data entry and spreadsheet organization
- Data validation and consistency checking
- Grouping data based on:
  - Location
  - Rating
  - Price range
  - Menu categories

---

## Tools Used

- Microsoft Excel
- Google Sheets
- Google Maps

---

## Dataset Columns

| Column Name | Description |
|---|---|
| No | Record number |
| Nama Tempat | Restaurant name |
| Kecamatan | District location |
| Menu | Food menu |
| Jumlah Ulasan Menu | Total menu reviews |
| Jumlah Total Ulasan | Total restaurant reviews |
| Rating | Restaurant rating |
| Harga Minimal | Minimum price |
| Harga Maksimal | Maximum price |
